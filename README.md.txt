# Perez Atelier - Site Vitrine

Site vitrine pour un service de climatisation et nettoyage automobile haut de gamme.  
Déployable sur GitHub Pages.

## Démarrage
- Ouvrez `index.html` dans un navigateur
- Modifiez le lien Calendly dans la section "contact"
- Déposez les fichiers sur GitHub pour publier

## Technologies
- HTML5, CSS3, JavaScript
- AOS.js (animation au scroll)
- Google Fonts : Poppins
